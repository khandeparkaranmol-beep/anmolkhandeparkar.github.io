◆ The Pipeline

A sequential 5-stage pipeline where each stage feeds directly into the next. The whole thing runs end-to-end in under 2 minutes.

Jina AI Reader converts any company URL — including React SPAs and Angular apps — into clean, parseable markdown. No brittle scrapers.
Jina AI

Claude API

Rule Engine

Claude API
05

3–5 PE funds, each with a plain-English explanation of why they're a fit — deal size, sector thesis, portfolio match, geography. Explainable, not just a list.
OUTPUT ✓
WHY HYBRID — NOT PURE LLM
Asking an LLM to reason across 500 funds at once is expensive, slow, and drifts. Hard rules eliminate the obviously wrong candidates first — then LLM reasoning handles the nuanced fit scoring. Rules for speed. LLM for judgment.
WHY TOOL CALLING FOR EXTRACTION
Free-form extraction hallucinates missing fields and returns inconsistent shapes. Schema enforcement via tool calling guarantees every company profile has the same structure — no downstream null checks, no format surprises. Deterministic inputs → reliable outputs.
